# healson-probe-capture (Ultra-sound video capture)

This part of the project allows the ultrasound images to be in the visual app
so that we can overlay the brain images with the robot so that needle insertion
is more realistic. 

Modify the IGTLink port 

Modify the captured area (pixels) to fit your screen

Run capture_probe.py 

It will advertise the frames on device "HealsonProbe" to be used in 3Dslicer.
